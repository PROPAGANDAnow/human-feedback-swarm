# Human Feedback Swarm

A rapid prototype for a human-powered UX research platform based on call notes with Bobby.

## Concept

Deploy a product or staging environment, brief a cohort of 50–100 real humans, collect in-browser screen-recorded feedback, verify authenticity with identity + agent QA, and turn raw evidence into decision-ready product insight.

## Prototype includes

- Single-page interactive product concept
- Researcher / tester / reviewer mode switcher
- Dynamic cohort size + payout model
- Product-native testing workflow mock
- Trust / verification / payout narrative

## Local development

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy

Designed for static deployment on Vercel.
